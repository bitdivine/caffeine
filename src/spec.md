# Simple Landing Page Application

## Overview
A minimal web application consisting of a single landing page that displays only the word "caffeine" centered on the screen.

## Features
- Single landing page
- Display the text "caffeine" centered on the screen
- No additional content, navigation, or interactive elements

## Backend Requirements
No backend functionality is required for this application.

## Frontend Requirements
- Single page displaying the text "caffeine"
- Text should be centered both horizontally and vertically on the screen
- No additional styling, navigation, or interactive elements
- Application language: English
